CSS Typography

In this chapter:

  • fonts
    • Font usage depends entirely on the users system. They can't print using a font they don't personally have, so webfonts aren't an option.
  • Font units: pt vs. px
    • Points were designed for print. Using px for print makes as little sense as using pt for web.
  • indents
  • line spacing
  • paragraph spacing
  • headings

Many of these principles apply to both web design and print design.

Sometime books indent all their paragraphs. Use text-indent for this. 

You can also outdent (eg, a bibliography) using margin-left and a negative text-indent.
