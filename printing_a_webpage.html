Printing a Webpage

Alright, so we have a webpage. Let's use my website version of this book as the example. We need to strip some stuff out that doesn't make sense for print. Then we'll get really fancy and make it look bookish. 

<LINK rel="stylesheet" type"text/css" href="screen.css" media="screen">
<LINK rel="stylesheet" type"text/css" href="print.css" media="print">
@media print {
  selector {
    styles;
  }
}

@media screen {
  selector {
    styles;
  }
}
