Indent All Paragraphs Except After Headings

A standard design practice is to indent all plain text paragraphs except for those that immediately follow a heading. We can use pseudoclasses to achieve this.

p {
  text-indent: 20pt;
}

h1 > p:first-of-type, 
h2 > p:first-of-type, 
h3 > p:first-of-type {
  text-indent: 0;
}
